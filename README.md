<ul>
  <li> <b>FEDBIOJEPA Framework:</b> Co-authored research on a self-supervised federated knowledge graph for computational drug repositioning. Includes a custom negative sampling pipeline designed to robustly corrupt both head and tail entities for improved graph representation learning.</li>
  <li> <b>Active Motion Tracking System:</b> A real-time pan-tilt face and hand gesture tracking system powered by a Raspberry Pi 4 (OpenCV/Mediapipe) and a Pico 2W.</li>
