01 — About

A bioengineer who ended up writing backends

I started in bioengineering, then moved into computer engineering — and kept the habit of asking what a system is actually measuring. That's shaped almost everything I build since: ML pipelines for genomic variant classification, an ILP solver for drug regimen conflicts, an LSTM watching patient time-series for thrombosis risk.

Outside of healthcare-flavored ML, I build the more ordinary engineering muscle: Go and FastAPI backends, CI/CD pipelines, and a small fleet of Raspberry Pi Pico robots that track faces, avoid walls, and map rooms with ultrasound.

Stroke Prediction System

ML

Compares seven classifiers across a 5,110-record clinical set and a 70k-record symptom set with 19.5:1 class imbalance. Deployed on Streamlit with SHAP explanations.

Stack: Python, XGBoost, CatBoost, Streamlit, SHAP

Clinical Drug Regimen Optimizer

Healthcare
